# The Selling App mockup
This is a mockup for online selling application with vue.js

## How to run
Start api-server
```
// in ./api-server
$ gem install foreman // if required
$ bundle install // if required
$ foreman start
```
Run front-end app
```
// in ./front-end
$ npm i
$ npm run build
```
